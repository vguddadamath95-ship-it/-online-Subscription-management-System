What is SubManager?

SubManager is an Online Subscription Management System built as a college micro project. It allows users to browse subscription plans, subscribe, manage their subscriptions, and track payment history.

This project demonstrates full-stack web development using Python Flask for the backend, SQLite for the database, and HTML/CSS/Bootstrap for the frontend.

Technologies Used:
Python Flask SQLite HTML5 CSS3 JavaScript Bootstrap 5
